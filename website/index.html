SimpleDropit

A simple drag-n-drop file upload vanilla js library, easy to use across modern browsers.

View SimpleDropit on Github

Usage

You can install SimpleDropit plugin by including the following stylesheet and javascript:

<link rel="stylesheet" href="dist/css/simpledropit.min.css">
<script src="dist/js/simpledropit.min.js"></script>

Example

Just call new SimpleDropit(ELEMENT) on the file-select input element.

Your input field will be

<input type="file" name="files[]" id="myInput" multiple>

And you start SimpleDropit manually:

new SimpleDropit(document.getElementById('myInput'));

Your input field html structure will be similar to this:

<div class="sd-box">
    <div class="sd-box-wrapper">
        <div class="sd-label-wrapper">
            <span class="sd-box-dragndrop">Drop file here / </span>
            <span class="sd-box-file-name"></span>
            <label class="sd-label">Browse <span class="sd-box-browse-file">File</span></label>
            <input type="file" name="files[]" id="myInput" multiple>
        </div>
    </div>
</div>

You can also use the html structure in your code and call SimpleDropit.

<div class="sd-box">
    <div class="sd-box-wrapper">
        <div class="sd-label-wrapper">
            <span class="sd-box-dragndrop">Drop file here / </span>
            <span class="sd-box-file-name"></span>
            <label class="sd-label">Browse <span class="sd-box-browse-file">File</span></label>
            <input type="file" name="files[]" id="myInput2" multiple>
        </div>
    </div>
</div>

<script type="text/javascript">
    new SimpleDropit(document.getElementById('myInput2'));
</script>
Drop file here / 

If you want to use jQuery:

new SimpleDropit($('#myInput')[0]);

Browser Compatibility

SimpleDropit works in all modern browsers.
